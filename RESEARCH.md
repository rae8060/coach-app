## 🏀 Basketball Coaching

### Youth Drills (Ages 6-13)
- **Dribbling**: Pirate Dribbling, Mr. Fox, Coin Drop
- **Passing**: Partner passing (teaches basics, different pass types)
- **Shooting**: Form shooting, lay-ups
- **Footwork**: Essential for all positions
- **Key skills**: dribbling, passing, shooting, lay-ups, footwork, game concepts

### Basic Offensive Plays (for our app)
1. **Give and Go** - Pass, cut to hoop, receive return pass
2. **Pick and Roll** - Screen for ball handler, roll to basket
3. **Motion Offense** - Pass, cut, fill (continuous movement)
4. **5-Out Motion** - All players can shoot, constant movement
5. **4-Out 1-In** - 4 players outside, 1 inside

### Defense Positioning
- Help defense principles
- Closeout technique
- Man-to-man basics
- Zone defense introduction

## 🎯 Teaching Basketball Reads & Reactions (NEW!)

### Read & React Offense
- **What it is:** Teaches players HOW to play, not robotic pre-set actions
- **Why it works:** Players read defense and react accordingly
- **Youth-friendly:** Can be taught in layers, starting with 3-player drills
- **Key principle:** Pass, cut, fill - continuous motion

### Teaching "Reads" to Kids
- Start with simple decisions
- Ask "what would you do here?"
- Build confidence in reading help defense
- Teach players to see the whole court

### Progressive Learning
1. Basic cuts and passes
2. 3-player drills
3. Add filling players
4. Full 5-on-5
